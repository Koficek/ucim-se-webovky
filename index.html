Co je HTML?


HTML (hypertextový značkovací jazyk) je psán mezi značkami, určujícími jeho význam.

Co je CSS


CSS označujeme jako kaskádové styly, upravuje vizuální stránku HTML.

Co je JavaScript?


JavaScript je jazyk pro zajištění interaktivity stránek.

HTML

HyperText Markup Language je základním stavebním kamenem webu. Umožňuje nám definovat strukturu obsahu a tvoří kostru každé moderní stránky.
